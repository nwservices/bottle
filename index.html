Bottle

A minimalistic css framework for web development.

I love bootsrap css, foundation css and even pure css, but what i found is they are little heavy for most of my applications and have become very much routeen in most web apps and so i got bored. Here is my css sheet which is minimal, im trying to keep only frequently used elements. Try it and do give your feedback :) This code is licenced under GNU GPL V2.
